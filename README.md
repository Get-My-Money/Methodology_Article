# Сравнительный анализ производительности YOLOv8 и YOLOv10 для робота-следователя на CPU

Данный проект посвящен сравнительному исследованию двух современных архитектур детекции объектов — **YOLOv8** (стабильный стандарт) и **YOLOv10** (архитектура без NMS) — в специфических условиях работы сервисного робота.

## Цель исследования
Оценить пригодность моделей для работы в реальном времени на бюджетном оборудовании (CPU) и их устойчивость к изменению освещения.

## Датасет и сценарии
Тестирование проводится на трех видеопотоках (45 секунд каждый), имитирующих типичные условия эксплуатации:

1. **`backlight.mp4`** — Контровой свет (окно за объектом). Критичный сценарий, где человек превращается в темный силуэт.
2. **`fulllight.mp4`** — Идеальное комнатное освещение. Базовый уровень для замера максимального FPS.
3. **`lowlight.mp4`** — Низкая освещенность. Проверка работы детектора при высоком уровне цифрового шума.

## Метрики
Помимо классических показателей, введено понятие **TTL (Time-To-Loss)**:
* **FPS (Frames Per Second)** — скорость инференса на CPU.
* **TTL (Time-To-Loss)** — длительность непрерывной потери цели. Измеряется количеством кадров подряд, на которых человек не был обнаружен. Это ключевой параметр безопасности для автономного следования.

## Технологический стек
* **Модели:** YOLOv8n (baseline) и YOLOv10n (NMS-free).
* **Оптимизация:** [OpenVINO Toolkit](https://www.intel.com/content/www/us/en/developer/tools/openvino-toolkit/overview.html) для ускорения работы на процессорах Intel.
* **Библиотеки:** `ultralytics`, `openvino`, `opencv-python`, `pandas`, `matplotlib`.

## Инференс (Пример для YOLOv10)

Модель YOLOv10 исключает этап Non-Maximum Suppression (NMS) на стороне CPU, что потенциально снижает задержки. Экспорт и запуск:

```python
from ultralytics import YOLO
import openvino as ov

# Экспорт модели
model = YOLO('yolov10n.pt')
model.export(format='openvino', imgsz=640)

# Загрузка в OpenVINO Runtime
core = ov.Core()
compiled_model = core.compile_model('yolov10n_openvino_model/yolov10n.xml', 'CPU')
output_layer = compiled_model.output(0)
